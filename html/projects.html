Research

Software Engineering for Systems

Modern distributed systems are difficult to reason about. We have developed a set of projects that make the process of instrumenting, debugging, and visualizing distributed systems simpler.

  • Distributed Clocks: a vector clock instrumentation library
  • Dinv: a distributed invariant checker (SOSP 2015 Poster)
  • ShiViz/TSViz: visualization engines for distributed executions (ICSE 2014, ICSE 2017)
  • PGo: A PlusCal spec to Go implemention compiler (SOSP 2017 ACM Student Research Champion)

Networked Datacenter Systems

We investigate new design principles in rack scale datacenter design, including VDC allocation, memory sharing, and disaggregation.

  • Decibel: a system for storage sharing in disaggregation (NSDI 2017)
  • Programmable fault tolerance and fate sharing in disaggregation (HotNets 2017)
  • NetSolver: a virtual data center (VDC) job allocation tool (ICICS-ECE-IEEE Workshop 2017 Poster)
  • BlueBridge: Global shared memory in rack scale deployments

Privacy-Preserving Data Systems

We leverage modern privacy-preserving methods to provide new guarantees to current system design paradigms. Privacy is preserved by maintaining computation at local sites, without any sharing of data. Additional privacy guarantees are provided through usage of differential privacy, onion routing (Tor) and key encryption.

  • InsuLearn: a system for building shared models across privately owned medical data (MICCAI MLMI 2017)
  • TorMentor: an anonymous messaging system for collaborative machine learning (In Submission)

Security

Computer security is of paramount importance. We enable user functionality and protect user data through a variety of methods.

  • TrustedCapsules: Programmable access control extending beyong system boundaries (In Submission)
  • Beeswax: A client-side platform for private web applications (PETS 2016)
  • Micasa: Preserving user data functionality after service end-of-life in hosted applications (SoCC 2013)
